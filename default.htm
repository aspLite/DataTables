How to use DataTables with classic ASP/VBScript?

"DataTables is a plug-in for the jQuery Javascript library. It is a highly flexible tool, built upon the foundations of progressive enhancement, that adds all of these advanced features to any HTML table." (source: https://datatables.net/)

Default.htm holds an Ajax-driven DataTable. Recordset.asp flushes the JSON it takes to build the DataTable. Datatables.asp holds a class with reusable code. aspLite is used for its JSON-features and some other general functions. Everything is wrapped in Bootstrap 5. Have fun!
